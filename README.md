# n8n Deployment & Workflows

## Overview
This repository contains scripts, configurations, and workflows to deploy and manage [n8n](https://n8n.io/) automations. It aims to simplify setup and provide reusable workflows.

## Features
- Deployment scripts for Docker / Docker Compose / WSL2
- Sample n8n workflows for automation
- Instructions for workflow import/export

## Prerequisites
- Windows 11 / Linux / macOS
- Docker & Docker Compose installed
- WSL2 (for Windows users)
- Git
